# 🚀 GUÍA RÁPIDA DE SETUP (5 PASOS)

## ⏱️ TIEMPO TOTAL: 10 MINUTOS

---

## PASO 1: ABRIR NOTION (1 min)
1. Entrá a notion.so
2. Si no tenés cuenta, creá una (gratis)
3. Creá una nueva página
4. Nombrala: "NTICx PRO"

---

## PASO 2: COPIAR EL DASHBOARD (3 min)
Abrí el archivo `nticx_pro_notion.md` (en la raíz de este proyecto) y copiá la sección **DASHBOARD PRINCIPAL**

Pegalo en tu página de Notion.

**Personalizá:**
- Cambiar "[Actualizar cada semana]" por el tema actual
- Cambiar el nombre si querés
- Agregar foto de perfil si querés hacer más visual

---

## PASO 3: CREAR LA BASE DE DATOS (3 min)


1. En Notion, escribí: `/database`
2. Seleccioná: "Table - Inline"
3. Llamala: "Misiones"

**Agregar estas columnas (propiedades):**

Click en "+" al lado de "Name"

Agregar una por una:

| Nombre | Tipo | Opciones |
|--------|------|----------|
| Semana | Number | - |
| Tipo | Select | Aprender, Practicar, Crear, PRO |
| Estado | Select | 🔒 Bloqueada, ⚡ Disponible, 🔄 En Progreso, ✅ Completada |
| Dificultad | Select | Fácil, Media, PRO |
| Tema Clase | Select | Ver lista en el archivo principal |
| Nivel | Number | - |

---

## PASO 4: CARGAR LAS MISIONES (2 min inicial)

**Opción A - Rápida (recomendada para empezar):**
Cargá solo las primeras 3 misiones (Semana 1)
Después vas agregando semana a semana

**Opción B - Completa:**
Cargá las 36 misiones de una (lleva 20 min)

**Cómo cargar una misión:**
1. Click en "+ New" en la base de datos
2. Completar los campos (Nombre, Semana, Tipo, etc.)
3. Abrir la página de la misión
4. Copiar el contenido del archivo (🎯 Objetivo, ⚡ Misión, etc.)
5. Pegarlo dentro

**Shortcut:**
Creá una plantilla con la estructura
Así solo completás el contenido

---

## PASO 5: COMPARTIR CON TU HIJA (1 min)
1. Click en "Share" (arriba a la derecha)
2. Agregar el email de tu hija
3. Darle permisos: "Can edit"
4. Enviar invitación
5. ¡Listo!

---

## ✅ CHECKLIST FINAL

Antes de empezar, verificá:

- [ ] Dashboard visible
- [ ] Base de datos creada con todas las propiedades
- [ ] Al menos 3 misiones cargadas (Semana 1)
- [ ] Tu hija tiene acceso
- [ ] Primeras 3 misiones en estado "⚡ Disponible"

## 🔄 RUTINA SEMANAL

### VIERNES (5 min):
1. Preguntarle: "¿Qué vieron esta semana?"
2. Actualizar "Tema actual" en el Dashboard
3. Desbloquear las 3 misiones de la semana
   (cambiar estado de "Bloqueada" a "Disponible")

### DOMINGO (5 min):
1. Revisar qué misiones completó
2. Dejar comentarios o emojis 🎉
3. Actualizar el % de progreso
4. Planear siguiente semana juntos
