# 🌌 Proyecto Antigravity - v1.0 (Autonomous Intelligence Stack)

Un ecosistema de software soberano diseñado para la extracción, aprendizaje y reportería autónoma en entornos Windows.

---

## 🟢 Estado del Sistema: Nivel 10 - Operativo (Full Cycle)

Antigravity es una solución de inteligencia autónoma que orquesta múltiples "skills" para realizar tareas complejas de recolección de datos, análisis de causa raíz y generación de informes ejecutivos sin intervención humana.

---

## 🏛️ Arquitectura (Directivas Operativas)

El sistema se rige por 10 directivas críticas que garantizan su soberanía técnica:

1.  **D001_Indexador**: Auto-diagnóstico y validación del mapa del sistema.
2.  **D002_Logger**: Trazabilidad inmutable con persistencia absoluta (Fix A01).
3.  **D003_Brain**: Sistema de aprendizaje y diagnóstico de errores.
4.  **D004_Scraper**: Extracción automatizada de información.
5.  **D005_Reporter**: Generación de reportes ejecutivos Markdown.
6.  **D006_Orquestador**: Control central del flujo de misión.
7.  **D007_Comunicador**: Alertas internas y flujo de mensajes.
8.  **D008_Automatizacion**: Monitor de salud (Heartbeat) y ejecución continua.
9.  **D009_Notificaciones**: Integración con canales externos (Discord/Slack).
10. **D010_Mantenimiento**: Sistema inmunológico (Purga de logs y archivado).

---

## 🛠️ Stack Técnico

- **Lenguaje**: Python (invocado vía `py` para máxima compatibilidad Windows).
- **Control de Versiones**: Git-Ops (Sincronización automática con repositorio remoto).
- **Automatización**: Windows Batch (.bat) y Task Scheduler.
- **Entorno**: PowerShell / CMD.
- **Configuración**: Gestión de secretos mediante archivos `.env`.

---

## 🚀 Modo de Uso

Para disparar el ciclo autónomo completo de Antigravity (Mantenimiento -> Extracción -> Aprendizaje -> Reporte -> Notificación -> Health Check), simplemente ejecuta el lanzador principal:

```powershell
./antigravity_run.bat
```

### Automatización Pro
Para una autonomía total, se recomienda configurar **Windows Task Scheduler** para ejecutar `antigravity_run.bat` con una frecuencia diaria o semanal.
